# Amazon Listing Rules

## Title
- Maximum 200 characters (some categories 80)
- Format: Brand + Model/Series + Product Type + Key Features + Size/Color/Quantity
- Capitalize first letter of each major word
- Include size, color, material if applicable
- Do NOT include price, promotional phrases ("best seller"), subjective claims
- Example: "Handmade Ceramic Coffee Mug, 12 Oz, Minimalist Design, Dishwasher Safe, Blue"

## Bullet Points (Key Features)
- 5 bullet points maximum
- Each bullet: 200 characters max
- Start with capital letter, no period at end
- Lead with benefit, follow with feature
- Example: "DISHWASHER SAFE – Glazed interior and exterior withstand daily machine washing without fading"

## Description
- 2000 characters max
- HTML allowed: <br>, <b>, <ul>, <li>
- Focus on use cases, benefits, and customer reassurance
- Include relevant keywords not already in title/bullets

## Backend Keywords
- 250 bytes total
- Do not repeat words already in title/bullets
- Include common misspellings, synonyms

## Pricing
- Factor Amazon's 15% referral fee into pricing
- Consider FBA fees if shipping via Amazon warehouse
