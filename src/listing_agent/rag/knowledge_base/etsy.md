# Etsy Listing Rules

## Title
- Maximum 140 characters
- Etsy's algorithm heavily weights the first few words
- Use buyer-search language: what would someone type to find this?
- Include handmade, vintage, or material descriptors — Etsy buyers expect them
- Example: "Handmade Ceramic Mug | Minimalist Coffee Cup | 12oz | Gift for Coffee Lovers"

## Description
- No limit but 160 characters appear in Google snippets
- Start with who this is for and what makes it special
- Include keywords: handmade, ceramic, mug, coffee, gift
- Add dimensions, materials, care instructions
- Mention customization options if available

## Tags
- 13 tags maximum, each up to 20 characters
- Use all 13 — each is a search opportunity
- Mix specific ("ceramic coffee mug") and broad ("handmade gift")
- Include occasion tags: "housewarming gift", "birthday gift"
- Etsy buyers often search phrases, not single words

## Pricing
- Research competitor pricing before setting price
- Etsy charges 6.5% transaction fee + listing fee $0.20
- Offer free shipping (Etsy promotes these listings in search)
- Consider quantity discounts for repeat buyers

## Photos
- First photo is crucial (thumbnail)
- Show scale (hand holding the mug)
- Lifestyle shots outperform studio shots for handmade goods
