# Shopify Listing Rules

## Title
- Maximum 255 characters
- Put primary keyword in first 70 characters (visible in search previews)
- Include brand, product type, key differentiator
- Avoid ALL CAPS, excessive punctuation
- Example: "Handmade Ceramic Coffee Mug | 12oz | Dishwasher Safe | Minimalist Design"

## Description
- No character limit but 150-300 words is optimal for SEO
- First 150 characters appear in search snippets — make them count
- Use bullet points for features, paragraphs for story/brand
- Include keywords naturally: mug, coffee mug, ceramic mug, handmade mug
- Add care instructions for physical products

## Tags
- Maximum 250 characters total across all tags
- Tags are comma-separated
- Use long-tail keywords: "handmade ceramic mug", "minimalist coffee cup"
- Include material, color, style, use case tags

## Pricing
- Show MSRP clearly
- Handmade products: factor in 2-3x material cost + hourly rate
